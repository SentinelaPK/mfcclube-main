# Security Policy

## Supported Versions

As seguintes versões do projeto recebem atualizações de segurança:

| Version | Supported |
| ------- | --------- |
| 5.1.x   | ✅ Yes    |
| 5.0.x   | ❌ No     |
| 4.0.x   | ✅ Yes    |
| < 4.0   | ❌ No     |

Recomendamos fortemente que todos os usuários utilizem uma versão suportada para garantir correções de segurança e estabilidade.

## Reporting a Vulnerability

Levamos a segurança deste projeto a sério. Caso você identifique uma possível vulnerabilidade, siga as orientações abaixo:

### Como reportar
- **Não** abra issues públicas para falhas de segurança.
- Envie um e-mail para: **mentirafc.official@gmail.com**  
 

### O que incluir no relatório
- Descrição detalhada da vulnerabilidade
- Passos para reproduzir o problema
- Versão afetada do projeto
- Evidências relevantes (logs, screenshots, PoC, se possível)

### Processo de resposta
- Você receberá uma confirmação de recebimento em até **72 horas**
- Avaliaremos o relatório e informaremos se a vulnerabilidade foi:
  - ✅ Aceita e corrigida
  - ❌ Considerada fora de escopo ou inválida
- Caso aceita, trabalharemos em uma correção e informaremos quando ela estiver disponível

Agradecemos sua colaboração para manter este projeto seguro.
